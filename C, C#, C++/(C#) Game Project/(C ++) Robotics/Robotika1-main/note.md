Problema:


Ištestuoti, kaip veikia Ultragarso, Micro Servo senosoriai. Jų informaciją pateikti skirtinguose LCD ekranuose, taip pat pagal servo laipsnį RGB LED keisti spalvą.

Dizainas ir veikimas:

Programa nuolat matuoja atstumą ultragarsiniu jutikliu, rodo jį LCD ekrane, judina servo variklį pirmyn ir atgal, o servo kampui keičiantis keičia RGB LED spalvą (raudona, žalia, mėlyna).

Papildomi komentarai:

Iš pradžių norėjau padaryti, kažką panašaus į radaro sistemą, tačiau per tinkercad, niekaip neišėjo sujungti Servo su Ultragarso sensoriu (kad garso sensorius sukiotųsi). Realybėje, tai padaryti nebūtų sunku reiktų pritvirtinti garso sensorių prie Servo ir būtų pagrindinė mintis – primityvus radaras, įvykdyta.
